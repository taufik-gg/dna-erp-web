AI Layer - Claude Integration

Apa itu AI Layer?

AI Layer adalah integrasi Claude (Anthropic) ke dalam sistem DNA yang memungkinkan user mengubah konfigurasi menggunakan bahasa manusia (natural language).

Claude AI bertindak sebagai "translator" yang mengubah perintah manusia menjadi konfigurasi teknis yang dapat dipahami sistem.

AI Layer digunakan di:
  • ‚Ä¢ DNA Rules Editor - Edit approval thresholds, SLA, settings
  • ‚Ä¢ Layout Editor - Generate komponen via chat

Contoh Perintah AI:

Rules Editor:
"Ubah threshold manager menjadi 5 juta"
Rules Editor:
"Kurangi SLA director jadi 24 jam"
Layout Editor:
"Tambahkan 3 summary card untuk status PO"
Layout Editor:
"Tambahkan pie chart dan bar chart"

üé® UI Automate Flow:

1. User drag & drop komponen di Layout Editor
2. (Opsional) Gunakan AI chat untuk generate komponen
3. Save ‚Üí Auto-generate Markdown + JSON
4. Commit ke GitHub ‚Üí Webhook trigger
5. Orchestrator generate layout-config.ts
6. ERP rebuild ‚Üí UI ter-render otomatis!

ü§ñ AI Layer Flow:

1. User ketik perintah bahasa manusia
2. Claude AI parse & translate ke config
3. Preview perubahan di panel kanan
4. User klik "Terapkan"
5. Auto-save ke DNA Repo + Git push
6. Orchestrator ‚Üí ERP auto-rebuild!
